🔥 Termux Ultimate Setup Tool

A powerful Python script for automatically setting up Termux with essential packages and tools.

---

✨ Features

- 📦 Auto update & upgrade
- 🐍 Python & pip installer
- 🌐 Networking tools
- 🛠️ Development utilities
- 🧰 Basic Termux setup
- ⚡ One-click installation

---

📥 Installation

pkg update && pkg upgrade -y

pkg install git -y

git clone https://github.com/xzhasan/Termux--update.py.git

cd update.py

python update.py
